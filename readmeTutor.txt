Untuk template atau view ada di folder resources/views
untuk model atau operasi yang membantu pemodelan table pada database ada di folder app/model
untuk controllers, ada difolder app/Http/Controllers
untuk menambahkan css, js, img, library dari luar, bisa diletakkan di public
untuk routing, dilakukan di folder routes/web.php
gunakan collection untuk mempermudah operasi pada models (ada dokumentasi dari laravel)
